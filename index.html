How to use AVGVSTO Simulator:

  • Select USB: Choose a USB key from the dropdown and click "Set Secure USB Drive".
  • Select File: Click on a file in the Desktop area to select it.
  • Encrypt: With a file selected and USB set, click "Encrypt Selected". Enter a password. The file will be marked as encrypted (.avgvsto).
  • Decrypt: Select an encrypted file (.avgvsto) and click "Decrypt Selected". You must use the same password that was used to encrypt it, along with the same USB key.
  • Security Rule: A file encrypted with USB_KEY_1 can only be decrypted with USB_KEY_1. Same for USB_KEY_2.
  • Note: This is a simulation. Files are not actually modified on your computer.
